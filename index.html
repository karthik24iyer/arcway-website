Why Arcway?

No VPN, no config

The relay handles connectivity. No port forwarding, no SSH tunnels.

Real PTY sessions

Colors, scrollback, interactive programs — exactly like your desktop terminal.

JWT + TLS secured

Only authenticated clients connect. Nothing is exposed publicly.

Self-hostable

Open-source relay server. Run it on any machine you own.

Self-host your relay

The relay server is open-source. Run it on any machine — your own server, a VPS, or a home machine behind a domain.

01

Clone & install

# Requires Node.js 18+ git clone https://github.com/karthik24iyer/arcway-relay-server cd arcway-relay-server && npm install
02

Set environment variables

# Required JWT_SECRET=$(openssl rand -base64 32) DATABASE_URL=postgresql://user:pass@localhost/arcway GOOGLE_CLIENT_ID=your-google-oauth-client-id   # Optional PORT=8080 ALLOWED_ORIGINS=https://yourdomain.com
03

Run migrations & start

node migrate.js npm start   # Server listens on 0.0.0.0:8080
04

Point the apps to your server

In the Arcway app settings, enter your relay server URL (e.g. wss://yourdomain.com) to connect to your own instance instead of the default.

Requirements
  • Node.js 18+
  • PostgreSQL 14+
  • A domain with TLS (required for WSS)
  • Google OAuth client ID
Quick deploy options
  • Any Linux VPS (DigitalOcean, Hetzner)
  • Railway / Render (free tier available)
  • Home server behind Cloudflare Tunnel
